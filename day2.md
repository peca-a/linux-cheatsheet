# 📅 Day 2 – Advanced Linux

------------------------------------------------------------------------------------------------

## 🔐 Permissions

### r w x

r → read
w → write
x → execute

------------------------------------------------------------------------------------------------

### Primer

-rw-r--r--

* vlasnik: read + write
* grupa: read
* ostali: read

------------------------------------------------------------------------------------------------

### chmod (oktalni)

chmod 644 file.txt
chmod 755 file

------------------------------------------------------------------------------------------------

### chmod (simbolički)

chmod u+x file
chmod go+r file
chmod a-x file

------------------------------------------------------------------------------------------------

### direktorijumi

r → listanje fajlova
x → ulazak u direktorijum
w → dodavanje/brisanje fajlova

------------------------------------------------------------------------------------------------

## 🔗 Links

### simbolički link

ln -s file.txt link.txt

* prečica ka fajlu

------------------------------------------------------------------------------------------------

### hard link

ln file.txt link.txt

* isto ime istog fajla

------------------------------------------------------------------------------------------------

### razlika

soft link → zavisi od originala
hard link → ne zavisi

------------------------------------------------------------------------------------------------

## 🔧 sed (osnovno)

sed 's/old/new/' file

* menja prvo pojavljivanje

sed 's/old/new/g' file

* menja sva pojavljivanja

------------------------------------------------------------------------------------------------

### regex

^ → početak linije
$ → kraj linije

* → ponavljanje

------------------------------------------------------------------------------------------------

## 🔗 alias

alias ll='ls -l'
unalias ll

------------------------------------------------------------------------------------------------

## 📦 tar

tar cvf arhiva.tar folder
tar czvf arhiva.tar.gz folder
tar xvf arhiva.tar

------------------------------------------------------------------------------------------------

## 🔍 diff

diff file1 file2
diff -r dir1 dir2

------------------------------------------------------------------------------------------------

## 💾 Disk

du -sh folder
df -h

------------------------------------------------------------------------------------------------
